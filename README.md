This project intends to study the link between local economic issues such as unemployment and their mobilization in political discourse. 
We also wish to check if this relation differs along political parties.
The corpus of texts we study are political manifestos from 1988 and 1993 legislative elections. 


The order in which the codes should be ran is the following : 
- ocr_retrieving/ocr_retrieval.py
- ocr_retrieving/mapping_ocr.py
- NLP/text_cleaning.py
- ocr_retrieving/na_cleaning.py

- unemployment_data/id_codes.py
- unemployment_data/retrieving_series.py
- unemployment_data/final_cleaning.py

- NLP/LDA.py
- NLP/BERTopic.py
- LDA_Study.py

- correlation/building_features.py
- correlation/correlation_analysis.py
